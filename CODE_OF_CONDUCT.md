# Code of Conduct

## Our Standard

Be respectful, constructive, and generous with context.

We want this project to feel welcoming for first-time contributors and reliable for maintainers. Harassment, abuse, or bad-faith participation are not acceptable.

## Enforcement

If you experience or observe a serious problem, report it through the process in [`SECURITY.md`](./SECURITY.md) or by opening a private maintainer contact request.
